ส่วนเสริม

เป็นสิ่งที่ปรับแต่งในเซิร์ฟเวอร์ และ หน้านี้จะอธิบายว่ามีอะไรบ้าง เปลียนอะไรไป (ไม่เกิน Vanilla)

Datapack

Bamboo Buff

เพิ่มสูตรไม้ไผ่ใน Cutting Stone ให้สามารถใช้งานได้ง่ายขึ้น

Plugin

Death Chest / Location

เมื่อคุณตายแล้วของคุณจะถูกเก็บไว้ในกล่องผู้เล่นอื่นไม่สามารถเปิดได้ และ พิกัดที่คุณตายจะส่งไปให้คุณในแชท

Plugin

Squaremap

เป็นโปรแกรมดูแผนที่โลกที่เรียบง่ายและเบาสำหรับเซิร์ฟเวอร์ Minecraft

Plugin

AnarchyStats

เป็นปลั๊กอินง่ายๆ ที่จะแสดงอายุเซิร์ฟเวอร์ ผู้เล่นทั้งหมด และขนาดโลกด้วยคำสั่ง /info

Plugin

CoreProtect

เครื่องมือบันทึกข้อมูลลง Database SQL และต่อต้านเกรียนที่รวดเร็ว มีประสิทธิภาพ ย้อนกลับและกู้คืนความเสียหายจำนวนเท่าใดก็ได้ ออกแบบโดยคำนึงถึงเซิร์ฟเวอร์ขนาดใหญ่ CoreProtect จะบันทึกและจัดการข้อมูลโดยไม่ส่งผลกระทบต่อประสิทธิภาพเซิร์ฟเวอร์

Plugin

DiscordSRV

คุณจะสามารถให้ผู้เล่นสามารถแชทในเกมเพื่อแชทกับผู้เล่นบนเซิร์ฟเวอร์ Discord ของคุณ รวมทั้งให้ผู้คนในเซิร์ฟเวอร์ Discord สามารถแชทกับผู้คนบนเซิร์ฟเวอร์ได้

Plugin / Mod

ItemSwapper

จุดอ่อนของคลังตามวิดีโอของXisumavoid เพิ่มอินเทอร์เฟซการสลับรายการที่ทริกเกอร์โดยการกดปุ่มลัด (ค่าเริ่มต้น: R)

Plugin / Mod

Simple Voicechat

ม็อดนี้เพิ่มการแชทด้วยเสียงที่ใกล้เคียงไปยังเซิร์ฟเวอร์ สามารถเลือกระหว่างการกดเพื่อพูด (PTT) หรือการสั่งงานด้วยเสียง คีย์ PTT เริ่มต้นคือ CAPS LOCK แต่สามารถเปลี่ยนตั้งค่าการโยงคีย์ สามารถเข้าถึงการตั้งค่าโดยกดปุ่ม V

Plugin

Reputation

คุณสามารถโหวตคะแนน ความชื่อถือผู้เล่นอื่นได้ด้วย การ /rep (ชื่อผู้เล่น)

Experimental

Update 1.20

1.20 เป็นการอัปเดตหลักที่กำลังจะมีขึ้นสำหรับ Java Edition ซึ่งจะออกในช่วงปี 2023 ดูเพิ่มเติมที่นี้

Experimental

Bundle

ใช้สำหรับจัดการไอเท็มภายในช่องเก็บของของผู้เล่น และอนุญาตให้รวมประเภทไอเท็มหลายประเภทภายในช่องไอเท็มเดียว
